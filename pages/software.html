Cheat Engine allows you to manipulate the in-game camera, time, and sometimes other features like the field of view (FOV). Keep in mind that Cheat Engine could very well trigger anti-cheat software used by games, so do your own research before attaching Cheat Engine to the process of a game with something like VAC, BattlEye or PunkBuster.


Note: We can't accept liability for banned accounts or other problems associated with running Cheat Engine.

It's called Cheat Engine but we are not using to cheat for gameplay mechanics!

Some Cheat Engine tables allow for the removal of the HUD. Other times, you’ll need to edit configuration files to get rid of the ugly bits that sit between the camera and your scene. Fortunately, jim2point0 has a Cheat Engine table for Infinite that includes toggles for the HUD along with God mode, slow motion, and a few other useful tweaks.

Here are two swapping photos we captured in Tomb Raider to demonstrate the drastic effect that Cheat Engine manipulation can have on the screenshot-taking process. The end result isn’t a masterpiece, but it’s a pretty clear departure from the screenshot taken at base FOV and the default camera.
